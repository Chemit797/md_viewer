## 第二部分：技术框架

### ADA（抗药抗体）形成链路与我们的切入位置

治疗性多肽进入体内后引发免疫反应的完整路径如下。**现有所有工具在[4]之前就停了，[7]是全球空白，这是我们的切入点。**

```
[1] 多肽进入体内
    ↓
[2] APC（树突细胞/巨噬细胞）吞噬多肽
    ↓
[3] APC内部cathepsin蛋白酶切割多肽为短片段
    ↓
[4] 片段装上MHC-II分子，呈递到细胞表面          ← NetMHCIIpan 覆盖到这里
    ↓
[5] CD4+ T细胞的TCR识别这个MHC-II+片段复合物
    ↓
[6] T细胞被激活                                 ← TLimmuno2 覆盖到这里（AUC 0.74）
    ↓
[7] T细胞分化：→ Tfh（促ADA）或 → Treg（抑ADA）  ← ★ 全球空白，我们的切入点
    ↓
[8] Tfh在生发中心辅助B细胞
    ↓
[9] B细胞产生ADA，影响药物疗效和安全性
```

**我们的预测目标：[7]，T细胞分化方向（Tfh vs Treg倾向），作为ADA风险的代理指标。**

---

### 为什么GEM能预测T细胞分化方向

这是整个项目的核心逻辑，有文献依据：

```
T细胞活化后发生代谢重编程（文献：Mayberry 2022, PMID 35493515）

活化后代谢偏向高糖酵解（Warburg效应）
    → HIF-1α激活，Bcl-6受抑制
    → 偏向Th17方向，ADA风险低

活化后代谢偏向低糖酵解 + 高脂肪酸氧化（FAO）
    → Bcl-6去抑制，CXCR5上调
    → 偏向Tfh方向，ADA风险高
```

**GEM（基因组尺度代谢模型）可以定量计算T细胞活化后各代谢通路的流量，即代谢状态的数字化表达。这是序列模型完全没有的信息维度。**

现有工具（NetMHCIIpan、TLimmuno2、EpiMatrix）只考虑多肽序列侧，没有任何T细胞代谢状态的特征。这是我们的差异化所在。

---

### 需要自己建的三个模型

整个预测系统由三个模型串联而成：

#### 模型1：Tfh专项GEM

| 项目 | 内容 |
|---|---|
| 是什么 | Tfh细胞内部代谢反应网络的数学地图，可以输入营养约束，输出各代谢通路的流量 |
| 为什么要建 | 现有GEM覆盖Th1/Th2/Th17/iTreg，**唯独没有Tfh**（Sen 2021, Cell Reports, PMID 34758307；Puniya 2021, NPJ Systems Biology，后者展望中明确写"未来希望扩展到Tfh"） |
| 怎么建 | 用tINIT算法，输入Tfh细胞scRNA-seq基因表达数据 + Human-GEM（全人类代谢网络），自动裁剪出Tfh特异性子网络 |
| 数据来源 | E-MTAB-13687（HCA人类细胞图谱扁桃体数据集，包含GC-Tfh细胞群，公开可获取）；如覆盖不足需补充数据 |
| 难度 | 中等偏上；最费时的环节是gap-filling（裁剪后孤立反应的填补与验证），预计2-3个月 |
| 发表价值 | 全球首个Tfh代谢模型，本身具备独立发表价值 |

#### 模型2：激活信号→代谢约束映射函数

| 项目 | 内容 |
|---|---|
| 是什么 | 把"多肽与MHC-II的结合强度"转化为"GEM的营养摄取约束"的函数 |
| 为什么要建 | NetMHCIIpan输出的是IC50（亲和力，nM），GEM需要输入的是葡萄糖摄取速率上界；两者之间的转化没有文献先例 |
| 怎么建 | 参考T细胞活化后葡萄糖摄取变化的已知范围（静息态→最大活化态），设计映射函数，通过消融实验比较不同形状（线性/指数/分段） |
| 最大风险 | **这是项目里不确定性最高的环节**，映射函数的形状直接影响GEM输出，且无任何文献可参照 |

#### 模型3：ML分类器

| 项目 | 内容 |
|---|---|
| 是什么 | 把GEM输出的代谢通量特征+序列特征映射到Tfh分化倾向分数的机器学习模型 |
| 输入 | 糖酵解/FAO/OXPHOS/谷氨酰胺各通路通量 + NetMHCIIpan亲和力分数 |
| 输出 | Tfh倾向分数（0-1），即ADA风险代理指标 |
| 训练标签 | 多肽PBMC体外实验的Tfh标志物（CXCR5+ PD-1high）或T细胞增殖SI值 |
| 关键依赖 | **训练标签数据。公开数据库无此类数据，必须依赖合作实验。这是整个项目的数据生死线。** |

### 完整Pipeline

```
治疗性多肽序列
    ↓
[借用] NetMHCIIpan：滑窗切割→每个15aa片段的MHC-II结合亲和力（IC50）
    ↓
[模型2] 映射函数：IC50矩阵 → GEM营养摄取约束条件
    ↓
[模型1] Tfh GEM + FBA求解：在约束下计算各代谢通路流量
    ↓
[模型3] ML分类器：通量特征 + 序列特征 → Tfh分化倾向分数
    ↓
输出：该多肽的ADA风险预测（高/低）
```

| 工具 | 能不能用 | 原因 |
|---|---|---|
| NetMHCIIpan-4.3 | ✅ 核心工具，调API | 设计上接受任意肽序列输入，已有将其用于exendin-4的先例（PMID 29758051） |

| scGPT/Geneformer | ⚠️ 备选方案用 | 不输出代谢通量，无法直接整合进GEM框架；保留为备选路线 |

## 第三部分：核心瓶颈

### 数据问题比预想严重

**数据情况总结：**

```
数据层级                              数量              可用性
────────────────────────────────────────────────────────────────
MHC-II结合亲和力数据（IEDB）           数万条             ✅ 可用于NetMHCIIpan
单抗（infliximab等）PBMC T细胞数据     几十~上百条        ⚠️ 可用于迁移学习起点
治疗性多肽PBMC T细胞激活数据           ≈ 0条              ❌ 全球公开库无
治疗性多肽临床ADA数据（结构化）         < 30条             ❌ 散落在临床报告中
```

**关键点：** 治疗性多肽的体外T细胞激活数据在全球公开数据库里几乎为零。这不是"数据少可以克服"的问题，而是"模型3的训练标签来源缺失"的问题。没有训练标签，ML分类器无法建立。

### 四条建模路线对比

在数据限制下，有四条可行路线：

| 维度 | 主线（GEM+ML） | 路线A（纯序列） | 路线B（scGPT） | 路线C（贝叶斯） |
|---|---|---|---|---|
| **数据需求** | 极高：多肽PBMC实验≥50对（必须） | 中：多肽标签≥30对 | 中：多肽标签≥30对 | 最低：文献公开数据够用 |
| **预计效果** | 理论上限最高（有T细胞侧机理特征） | 中等（无T细胞侧信息） | 中等偏上（有细胞状态信息） | 偏低（本质是表位密度后处理） |
| **整体难度** | ★★★★★ | ★★☆☆☆ | ★★★☆☆ | ★☆☆☆☆ |
| **发表上限** | Cell Systems / Nature Methods | Bioinformatics | NAR | 领域小刊 |
| **复旦合作** | 必须，是生死线 | 有帮助但非必须 | 有帮助但非必须 | 不需要 |

**主线与路线B的关键关系：**
两条路线共享前两步（NetMHCIIpan + scRNA-seq数据处理），切换成本极低。建议：主推主线，路线B作为应急通道。复旦数据充足→全力推主线；复旦数据延迟→路线B先出概念验证，买时间等数据。

**需要复旦提供的内容（按优先级）：**

### 决策3（请Alan判断）：5个免疫学机理问题

以下问题直接影响建模方案，需要免疫学专业判断，在组内无法自行解决：

---

**问题1：短肽能不能绕过APC加工直接呈递**

背景：完整ADA链路要求多肽被cathepsin切割成片段再装上MHC-II。但治疗性多肽本身是10-40aa，MHC-II结合槽容纳13-25aa，理论上短肽可能直接呈递而不需要切割。

影响建模的方式：
- 若必须切割 → 我们的滑窗方法（把长肽切成15aa片段分别预测）是合理的
- 若短肽可直接呈递 → 我们可以不滑窗，直接预测全长肽，精度更高

**需要判断：对于10-40aa的治疗性多肽，APC切割是必须发生的吗？**

---

**问题2：体外Tfh分化的可靠判定标准**

背景：我们需要体外实验数据作为训练标签。但"体外Tfh"的判定标准在文献里不统一。

标志物选择直接决定训练标签的定义，用了不可靠的标志物，整个ML训练的标签都是错的。

**需要判断：PBMC体外实验中，CXCR5+ PD-1high 够不够判定Tfh？还是必须加BCL-6蛋白表达？IL-21分泌能否单独作为readout？培养多少天可以看到稳定分化？**

---

**问题3：Tfh和Treg在代谢上真的能区分吗**

背景：这是整个项目GEM模块的核心假设。文献显示GC-Tfh是低糖酵解+高FAO，Treg也是低糖酵解+高FAO+高OXPHOS。两者在主代谢轴上高度相似。

如果代谢上区分不开Tfh和Treg，GEM通量特征就没有区分力，项目核心假设崩塌。

**需要判断：Tfh和Treg在代谢层面是否有可以被GEM建模的可区分差异？谷氨酰胺代谢、脂肪酸链长偏好、特定氨基酸消耗速率等维度是否有已知差异？**

---

**问题4：HLA多态性对治疗性多肽免疫原性的影响**

背景：不同HLA型的人对同一多肽免疫应答差异很大。我们的模型需要选择覆盖哪些HLA-DR等位基因。

**需要判断：面向中国人群，哪些HLA-DR等位基因对GLP-1类多肽的呈递贡献最大？体外实验供体应如何选择才能覆盖足够人群代表性？**

---

**问题5：体外T细胞激活数据对临床ADA的预测力**

背景：Walsh 2024（PMID 38863708）显示PBMC体外assay对单抗的预测特异性100%，敏感性只有53%。

这个数字决定了我们用体外readout作为训练标签的合理性。

**需要判断：对于治疗性多肽，体外T细胞增殖阳性和临床ADA之间的相关性如何？假阴性的主要原因是什么？**
